# 黑暗来临 (Dark Mode Toggle)

一款简单、美观且智能的 Chrome 浏览器黑暗模式扩展程序。它能瞬间将任何刺眼的亮色网页转换为舒适的暗色调，保护您的双眼。

## 🌟 核心特性

- **极致的反转算法**：利用高效的 CSS 滤镜 (`filter: invert`) 实现全局暗黑模式，并智能地将图片、视频、Canvas 等媒体内容二次反转回真实的原本色彩，保证内容浏览的体验。
- **原生深色智能检测**：在注入滤镜的同时，插件会自动侦测网页的主体结构是否已经自带了深色背景（例如开启了原生暗色模式的网站）。如果检测为原生深色，将自动撤销滤镜交还控制权，避免“负负得正”的二次反转闪烁。
- **灵活的黑白名单管理**：
  - **全局开启 (排除列表)**：默认所有网站开启黑暗模式。您可以随时将不想变黑的网站（遇到显示异常时）一键加入列表。
  - **仅限列表 (白名单)**：默认保持网站原始配色，只有当您将特定网站加入列表后，才会对该网站应用黑暗模式。
- **极简的扁平化界面**：参考 `ui-ux-pro-max` 规范打造的 Flat Design 弹窗界面。拥有柔和的阴影、高对比度的层级结构以及 150-300ms 的平滑交互微动画。
- **定制太极视觉**：专属的阴阳太极 (Yin Yang) 扩展图标，契合明暗交替的核心理念。

## 🚀 安装与使用

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 进入扩展管理页面，Edge 浏览器输入`edge://extensions/`。
2. 开启页面右上角的 **开发者模式**，Edge 浏览器在左侧栏中打开**开发人员模式**。
3. 将`dark-mode`文件夹拖入并松开。
4. 安装完成后，点击浏览器右上角的 **太极图标** 即可调出设置面板，随心切换网页模式！
